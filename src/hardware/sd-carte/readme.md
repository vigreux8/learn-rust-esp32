## Informations techniques de la carte SD

- **Transmission rapide des données** :  
  L'interface SPI est utilisée pour garantir une transmission rapide et stable des données, ce qui améliore l'efficacité du travail.

- **Forte compatibilité** :  
  Prise en charge des cartes Micro SD et Micro SDHC, répondant ainsi aux besoins de stockage de nombreux projets différents.

- **Circuit de conversion de niveau intégré** :
  - Le niveau de l'interface peut être de 5 V.
  - Tension d'alimentation de 5 V, avec circuit régulateur intégré.
  - L'interface de communication est une interface SPI standard.

- pin sur la carte :
  - cs
  - sck
  - mosi
  - miso
  - vcc
  - gnd
